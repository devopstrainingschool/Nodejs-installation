# Nodejs-installation

## Install NodeJS package manager(nmp)
```
curl -sL https://rpm.nodesource.com/setup_10.x | sudo bash -
```
```
sudo yum install nodejs
```
## verify the Node.JS and npm Installation
```
node --version
```
```
npm --version
```
